# Zaheer Ahmed - Personal Portfolio 🚀

Welcome to the repository of my personal portfolio website! This project showcases my skills, experience, projects, and achievements as a Full-Stack Developer. It features a modern, responsive design with smooth animations and a fully functional contact form.

## 🌟 Features

- **Modern & Responsive Design:** Built with a mobile-first approach ensuring the website looks stunning on all devices.
- **Dynamic Theming:** Seamless Dark and Light mode support with beautifully crafted color palettes.
- **Performant & Fast:** Developed using Next.js (App Router) and Turbopack for lightning-fast load times.
- **Working Contact Form:** Integrated with **Resend** to deliver messages directly to my inbox.
- **Custom Assets:** Unique UI components, SVG icons, and optimized WebP imagery.

## 🛠️ Tech Stack

- **Framework:** Next.js (React)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Email Service:** Resend API
- **Icons:** Iconify

## 🚀 Getting Started

If you want to run this project locally, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/zaheer-ahmed77/portfolio-website.git
cd portfolio-website
```

### 2. Install dependencies
You can use `npm`, `yarn`, or `pnpm`.
```bash
npm install
# or
pnpm install
```

### 3. Setup Environment Variables
Create a `.env.local` file in the root directory and add your Resend API credentials to enable the contact form:
```env
RESEND_EMAIL_API_KEY="your_resend_api_key"
SENDER_EMAIL="onboarding@resend.dev"
RECEIVING_EMAIL="your_email@example.com"
```

### 4. Run the Development Server
```bash
npm run dev
# or
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
